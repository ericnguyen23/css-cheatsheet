CSS Snippet Cheatsheet

Here are some useful snippets

Flexbox

To make a div a flexbox:

  div {
    display:flex;
  }

Variables

To create a variable:

    :root {
      --variable-name: value;
    }

Selector

To select a specific file type:

    a[href$='.css']::after {
      propert: value;
    }
